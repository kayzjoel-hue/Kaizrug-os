# Kaizrug OS v1 Showroom

This repository is a static showroom site for an early Kaizrug OS version.
It has no build step, no package manager, and no backend runtime.

## Active Public Files

- `index.html` - showroom homepage
- `css/main.css` - shared active styling
- `templates/studio.html` - linked studio collection page
- `docs/workshop.html` - linked workshop log page

Deploy the repository root as a static site.

## Archive

- `archive/legacy/` - unused placeholder assets and old helper files

## Private Notes

Sensitive business/payment notes are kept outside Git under `private/`, which is
ignored by `.gitignore`. Do not commit files from that folder.
